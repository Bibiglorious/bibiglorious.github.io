Sentiment Analysis of Amazon Video Game Reviews

Applied Natural Language Processing techniques to perform sentiment analysis on customer reviews. The text data underwent preprocessing steps such as tokenization, lemmatization, stopword removal, and special character filtering. Transformed processed text into numerical representations using TF-IDF and employed machine learning models (kNN and SVM) to classify sentiment into negative, neutral, and positive categories.
